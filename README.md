# MCQ Test Simulator (MAKAUT CA4)

A web-based MCQ test simulator inspired by the MAKAUT CA4 examination pattern.  
This project replicates the real CA4 experience with question navigation, answer tracking, scoring, and reset functionality.

---

## Features

- Random selection of 25 questions from a question bank
- Question navigation using sidebar buttons
- Next and Previous question navigation
- Answer persistence across questions
- Score calculation on submission
- Result overlay with final score
- Reset and new game functionality
- Clean and structured user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---


---

## How It Works

1. On page load, 25 random questions are selected
2. User navigates questions using sidebar or buttons
3. Selected answers are stored per question
4. On submission, answers are compared with correct options
5. Final score is displayed in an overlay
6. User can reset or start a new game

---

## Purpose

This project was created to:
- Practice JavaScript DOM manipulation
- Understand state management in frontend applications
- Simulate real-world MCQ exam systems
- Replicate MAKAUT CA4 exam behavior for learning purposes
